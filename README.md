### Brain-games

### Requirements:
  - NodeJs
  - Ubuntu or WSL (for Windows)

### Setup:
  1. git clone git@github.com:Pinkp0ny/frontend-project-44.git
  2. Enter on the command: cd frontend-project-44
  3. Enter on the command: make install
  4. enter the game from the list:
   - brain-even
   - brain-calc
   - brain-gcd
   - brain-progression
   - brain-prime
